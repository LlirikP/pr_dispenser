Запсук проекта:

0. Проверьте, что порт не занят

1. Склонируйте репозиторий 

git clone https://github.com/LlirikP/pr_dispenser

Перейдите в проект

cd pr_dispenser/

2. (Необязательно) Создайте .env в корне проекта (замените на реальные данные) либо оставьте дефолтные значения:

DB_URL=postgres://pruser:prpass@db:5432/prdb?sslmode=disable

PORT_AUTH=8080

DB_HOST=db

DB_PORT=5432

DB_USER=pruser

DB_PASSWORD=prpass

DB_NAME=prdb

3. Запустите сервис командой:

docker compose up --build

Если всё прошло успешно, вы увидите:  Server starting on port 8080
